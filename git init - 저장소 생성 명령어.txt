git init - 저장소 생성 명령어
git status - 저장소에 현재 파일 상태를 확인하느 명령어 
로컬 -> git add 파일 이름 - 준비 -> git commitg -m '최초저장' -> 스냅샷
git checkout 해쉬값 - 전에 저장했던 파일로 돌아가기
git checkout master - 현재로 돌아오기
